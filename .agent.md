# Agent 提示

这个文档是用来给 Agent （你）提供提示的，包含程序结构方案和一些 LLM 没有学到的知识。


## Dart语言变更

### Dot shorthands

Dart 3.10 引入了 dot shorthands 的语法糖，允许在类型明确的情况下省略类名，直接以点开头，例如

```dart
// 传统写法
MyEnum stat = MyEnum.foo;
Int id = Int.parse('123');

// dot shorthands 写法
MyEnum stat = .foo;
Int id = .parse('123');
```

特别地，类主构造函数可以写为`.new()`，例如`Uuid uuid = .new();`，但是你（Agent）不应使用。
